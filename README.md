# DocDoc Clinic:

A demo application **(Flutter)** to  to showcase and apply **CI/CD, Fastlane, Bloc and clean architecture** principles. It supports authentication, and fetch medical specializations beside their doctors list.

# Key Technologies
- Dart
- Flutter
- CI/CD
- Retrofit
- Bloc
- GetIt
- Cubit
- Shimmer
